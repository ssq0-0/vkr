💡 Что мы измеряем и зачем?

  • RPS (Requests Per Second) — сколько кадров система может обработать в секунду. Больше = лучше.
  • Avg Latency — среднее время обработки одного кадра. Меньше = лучше.
  • P95 Latency — 95-й перцентиль задержки. Показывает стабильность под нагрузкой.
  • Memory Arena — C++ выделяет память заранее большими блоками, избегая malloc/free на каждый кадр.
  • sync.Pool — Go переиспользует объекты, но GC всё равно периодически их проверяет.
